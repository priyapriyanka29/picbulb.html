# 💡 Bulb Toggle Project

## 📌 Overview

A simple web application that turns a bulb ON and OFF using user interaction.

## 🚀 Features    

* Toggle bulb image (ON/OFF)
* Interactive button control
                      
## 🛠️ Tech Used       

* HTML
* JavaScript

## ▶️ Run

Open picbulb.html in your browser

## ⚠️ Note

Basic project to demonstrate DOM manipulation.
